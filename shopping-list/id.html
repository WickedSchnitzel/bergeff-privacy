3. Ringkasan: Tidak Ada Data Pribadi yang Dikumpulkan

MiniSho: Simple Shopping List tidak mengumpulkan, menyimpan, mengirimkan, atau membagikan data pribadi apa pun.

Aplikasi ini beroperasi sepenuhnya secara offline. Tidak ada koneksi internet, tidak ada server eksternal, tidak ada data yang keluar dari perangkat Anda.

  • Daftar belanja yang Anda buat (judul, item, jumlah, tanda centang)
  • Pengaturan aplikasi yang Anda konfigurasikan (preferensi tampilan, urutan, tema)

Semua data ini disimpan secara eksklusif di perangkat Anda dan tidak pernah meninggalkannya.

Pengecualian – Input Suara Opsional: Saat Anda menggunakan fitur input suara opsional (tombol mikrofon), audio yang Anda ucapkan diproses oleh Speech Recognizer bawaan Android, yang dapat mengirimkan audio ke server Google. Aplikasi itu sendiri tidak menyimpan atau memproses data audio apa pun. Lihat Bagian 4 dan 5.

4. Fitur Berbagi Daftar (Berbagi Tautan)

Aplikasi ini menyertakan fitur opsional untuk membagikan daftar belanja dengan pengguna lain melalui tautan (misalnya, melalui aplikasi perpesanan pilihan Anda). Konten dari daftar yang dipilih (nama dan item-itemnya) dikodekan langsung ke dalam tautan itu sendiri — tidak ada data yang dikirimkan ke server mana pun yang dioperasikan oleh kami atau pihak ketiga; tautan dibuat dan diproses sepenuhnya di perangkat Anda.

Perangkat penerima mendekode tautan secara lokal untuk membuat daftar baru. Kami tidak memiliki akses ke konten tautan, tidak menyimpannya, dan tidak dapat melihat dengan siapa Anda membagikannya. Karena tautan berisi konten daftar lengkap, Anda bertanggung jawab penuh untuk memilih saluran transmisi yang tepercaya dan memutuskan dengan siapa Anda membagikannya.

5. Koneksi Internet

Aplikasi ini beroperasi sepenuhnya secara offline untuk fungsi utamanya. Tidak ada permintaan jaringan, sinkronisasi latar belakang, telemetri, atau analitik.

Pengecualian – Input Suara Opsional: Jika Anda menggunakan fitur input suara opsional, Speech Recognizer Android mungkin memerlukan koneksi internet untuk memproses kata-kata yang Anda ucapkan. Ini ditangani sepenuhnya oleh sistem operasi perangkat Anda dan layanan suara Google — bukan oleh Aplikasi. Anda selalu dapat memasukkan item secara manual.

6. Layanan Pihak Ketiga

Tidak ada analitik, iklan, SDK media sosial, penyimpanan cloud, atau prosesor pembayaran yang digunakan untuk fungsi utama.

Pengecualian – Google Speech Recognition (Android): Fitur input suara opsional menggunakan Speech Recognizer bawaan Android (biasanya Google). Audio dapat dikirim ke server Google. Tunduk pada Kebijakan Privasi Google. Aplikasi tidak menyimpan, mengakses, atau membagikan data audio apa pun. Input suara sepenuhnya opsional.

7. Tidak Ada Pelacakan atau Pembuatan Profil

Tidak ada pelacakan perilaku, pelacakan lokasi, pengenal perangkat, pola penggunaan, pembuatan profil pengguna, atau cookie.

10. Hak Anda Berdasarkan GDPR

  • Hak akses (Art. 15 GDPR): Tidak ada data pribadi yang diproses — tidak ada yang perlu diungkapkan.
  • Hak atas rektifikasi (Art. 16 GDPR): Tidak ada data pribadi yang disimpan.
  • Hak untuk dihapuskan (Art. 17 GDPR): Copot pemasangan Aplikasi untuk menghapus semua data yang disimpan secara lokal.
  • Hak untuk membatasi pemrosesan (Art. 18 GDPR): Tidak berlaku.
  • Hak atas portabilitas data (Art. 20 GDPR): Tidak berlaku.
  • Hak untuk keberatan (Art. 21 GDPR): Tidak berlaku.
  • Hak untuk menarik persetujuan (Art. 7(3) GDPR): Jika pemrosesan didasarkan pada persetujuan (input suara), Anda dapat menariknya kapan saja dengan tidak menggunakan fitur input suara.
