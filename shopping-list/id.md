## 3. Ringkasan: Tidak Ada Data Pribadi yang Dikumpulkan

MiniSho: Simple Shopping List tidak mengumpulkan, menyimpan, mengirimkan, atau membagikan data pribadi apa pun.

Aplikasi ini dirancang untuk berfungsi sepenuhnya secara offline. Aplikasi itu sendiri tidak membuat koneksi internet dan tidak memerlukannya untuk fungsi utamanya. Aplikasi tidak berkomunikasi dengan server eksternal mana pun dan tidak mengirim informasi apa pun ke luar perangkat Anda — tidak ke pengembang, tidak ke pihak ketiga mana pun, dan tidak ke layanan cloud mana pun.

**Pengecualian – Input Suara Opsional:** Aplikasi ini menyertakan fitur input suara opsional. Saat Anda memilih untuk menggunakannya (dengan mengetuk tombol mikrofon), audio yang Anda ucapkan diproses oleh Pengenal Ucapan bawaan Android, yang mungkin mengirimkan audio ke server Google. Aplikasi itu sendiri tidak menyimpan atau memproses data audio apa pun. Lihat Bagian 5 dan 6 untuk detailnya.

Satu-satunya data yang dibuat dan disimpan oleh Aplikasi adalah:

- Daftar belanja yang Anda buat sendiri (judul, item, jumlah, tanda centang)
- Pengaturan Aplikasi yang Anda konfigurasi (mis. preferensi tampilan, urutan, tema)
- Pengaturan alarm dan notifikasi yang Anda konfigurasi (waktu terjadwal, hari dalam seminggu, teks pesan notifikasi)

Semua data ini disimpan secara eksklusif di perangkat Anda menggunakan mekanisme penyimpanan lokal standar sistem operasi. Data ini tidak pernah meninggalkan perangkat Anda.

## 4. Fitur Berbagi Daftar (Berbagi Tautan)

Aplikasi ini menyertakan fitur opsional yang memungkinkan Anda membagikan daftar belanja dengan pengguna lain melalui tautan (mis. melalui aplikasi perpesanan pilihan Anda). Saat Anda menggunakan fitur ini, konten dari daftar yang dipilih (nama dan itemnya) dienkode langsung ke dalam tautan itu sendiri. Tidak ada data yang dikirimkan ke server mana pun yang dioperasikan oleh kami atau pihak ketiga mana pun — tautan dibuat dan diproses sepenuhnya di perangkat Anda.

Perangkat penerima mendekode tautan secara lokal untuk membuat daftar baru. Kami tidak memiliki akses ke konten tautan, tidak menyimpannya, dan tidak dapat melihat dengan siapa Anda membagikannya atau apa isinya.

Karena tautan berisi konten daftar lengkap, Anda bertanggung jawab penuh untuk memilih saluran transmisi yang tepercaya (mis. pesan pribadi) dan untuk memutuskan dengan siapa Anda membagikannya.

## 5. Koneksi Internet

Aplikasi tidak memerlukan atau menggunakan koneksi internet untuk fungsi utamanya. Aplikasi itu sendiri tidak membuat koneksi internet. Aplikasi beroperasi sepenuhnya secara offline. Tidak ada permintaan jaringan, tidak ada sinkronisasi latar belakang, tidak ada telemetri, dan tidak ada analitik dalam bentuk apa pun.

**Pengecualian – Input Suara Opsional:** Jika Anda menggunakan fitur input suara opsional, Pengenal Ucapan bawaan Android mungkin memerlukan koneksi internet aktif untuk memproses kata-kata yang Anda ucapkan. Ini sepenuhnya ditangani oleh sistem operasi perangkat Anda dan layanan suara Google — bukan oleh Aplikasi. Anda selalu dapat memasukkan item secara manual tanpa menggunakan input suara.

## 6. Layanan Pihak Ketiga

Aplikasi tidak mengintegrasikan layanan pihak ketiga mana pun untuk fungsi utamanya. Aplikasi tidak menggunakan analitik, jaringan periklanan, SDK media sosial, penyimpanan cloud, atau prosesor pembayaran.

**Pengecualian – Google Speech Recognition (Android):** Fitur Input Suara opsional menggunakan Pengenal Ucapan bawaan Android. Di sebagian besar perangkat Android, layanan ini disediakan oleh Google dan dapat mengirimkan audio yang Anda ucapkan ke server Google untuk diproses. Ini mungkin melibatkan transfer data ke negara ketiga di luar UE/EEA, khususnya ke Amerika Serikat. Google LLC bersertifikat di bawah Kerangka Kerja Privasi Data UE-AS. Ini tunduk pada Kebijakan Privasi Google: https://policies.google.com/privacy. Aplikasi tidak menyimpan, mengakses, atau membagikan data audio apa pun. Input suara sepenuhnya opsional — Anda selalu dapat mengetik item secara manual.

**Dasar hukum (GDPR):** Pemrosesan data audio melalui fitur input suara didasarkan pada persetujuan Anda yang diberikan secara bebas, yang diungkapkan dengan secara aktif menekan dan menahan tombol mikrofon. Anda dapat menarik persetujuan ini kapan saja hanya dengan tidak menggunakan fitur input suara.

## 7. Tidak Ada Pelacakan atau Pembuatan Profil

Aplikasi tidak melacak perilaku, lokasi, pengidentifikasi perangkat, atau pola penggunaan Anda. Tidak ada profil pengguna yang dibuat. Tidak ada cookie atau teknologi pelacakan serupa yang digunakan. Tidak ada pengambilan keputusan otomatis, termasuk pembuatan profil, yang terjadi.

## 8. Data yang Disimpan Secara Lokal di Perangkat Anda

Satu-satunya data yang pernah disimpan oleh Aplikasi adalah yang Anda masukkan secara eksplisit:

- Daftar belanja (judul, item) — disimpan di perangkat Anda untuk fungsi inti aplikasi
- Pengaturan dan preferensi Aplikasi — disimpan di perangkat Anda untuk mempersonalisasi pengalaman aplikasi
- Pengaturan alarm dan notifikasi (waktu terjadwal, hari dalam seminggu, teks pesan) — disimpan di perangkat Anda untuk fitur pengingat

Anda dapat menghapus semua data Aplikasi kapan saja dengan mencopot pemasangan Aplikasi atau menghapus data Aplikasi melalui pengaturan sistem perangkat Anda. Pengembang tidak memiliki akses ke data ini dan tidak dapat memulihkannya atas nama Anda.

## 9. Durasi Penyimpanan

Data yang disimpan secara lokal di perangkat Anda tetap di sana sampai Anda mencopot pemasangan Aplikasi atau menghapus data Aplikasi secara manual melalui pengaturan sistem perangkat Anda. Pengembang tidak memiliki akses ke data ini dan tidak menyimpannya di server mana pun.

Untuk data audio yang diproses oleh Google melalui fitur input suara opsional, durasi penyimpanan diatur oleh Kebijakan Privasi Google LLC: https://policies.google.com/privacy

Aplikasi tidak mengumpulkan data apa pun dari pengguna mana pun, termasuk anak-anak. Tidak diperlukan verifikasi usia, karena tidak ada data pribadi yang diproses pada titik mana pun.

## 11. Hak Anda Berdasarkan Hukum yang Berlaku (GDPR)

Meskipun Aplikasi tidak memproses data pribadi apa pun, hak-hak berikut diberikan kepada Anda berdasarkan Peraturan Perlindungan Data Umum (UE) 2016/679 (GDPR) dan undang-undang nasional yang setara. Anda dapat menggunakan hak-hak ini kapan saja dengan menghubungi kami:

- **Hak akses:** Karena kami tidak memproses data pribadi, tidak ada yang perlu diungkapkan.
- **Hak untuk perbaikan:** Kami tidak menyimpan data pribadi apa pun.
- **Hak untuk penghapusan** ("hak untuk dilupakan"): Semua data berada secara lokal di perangkat Anda dan dapat dihapus dengan mencopot pemasangan Aplikasi.
- **Hak untuk membatasi pemrosesan:** Tidak berlaku, karena tidak ada data pribadi yang diproses.
- **Hak atas portabilitas data:** Tidak berlaku, karena tidak ada data pribadi yang kami proses.
- **Hak untuk keberatan:** Tidak berlaku, karena tidak ada data pribadi yang diproses.
- **Hak untuk menarik persetujuan:** Jika pemrosesan didasarkan pada persetujuan (input suara), Anda dapat menariknya kapan saja hanya dengan tidak menggunakan fitur input suara.
- **Hak untuk mengajukan keluhan:** Anda berhak mengajukan keluhan kepada otoritas pengawas. Karena pengembang berbasis di Jerman, otoritas yang berwenang adalah Komisaris Negara Bagian untuk Perlindungan Data dan Kebebasan Informasi Baden-Württemberg (LfDI BW): https://www.baden-wuerttemberg.datenschutz.de. Daftar semua otoritas perlindungan data Jerman tersedia di: https://www.bfdi.bund.de — Anda juga dapat menghubungi otoritas pengawas di negara tempat tinggal Anda. Jika Anda berada di Indonesia, pengawasan perlindungan data pribadi saat ini dilakukan oleh Kementerian Komunikasi dan Digital (MOCD/Komdigi) berdasarkan Undang-Undang Perlindungan Data Pribadi (UU PDP). Otoritas pengawas independen yang mandiri akan dibentuk melalui Peraturan Presiden.
